# NASA Project 🌌

Это одностраничный сайт с адаптивным дизайном, слайдером фотографий, загружаемых через админку, и сортировкой изображений drag & drop.

## 🚀 Технологии

- **Backend**: Python 3.12, Django 5.2, MySQL
- **Frontend**: Bootstrap 5, Slick Slider, jQuery
- **Дополнительные пакеты**:
  - `django-filer` — управление изображениями
  - `easy-thumbnails` — создание миниатюр
  - `django-admin-sortable2` — сортировка записей перетаскиванием
  - `mysqlclient` — драйвер MySQL
  - `python-dotenv` — работа с переменными окружения

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/vladonsikos/nasa_project.git
cd nasa_project
```
### 2. Создание и активация виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
# или venv\Scripts\activate для Windows
```
### 3. Установка зависимостей
```bash
pip install -r req.pip
```
### 4. Настройка базы данных MySQL
Убедитесь, что MySQL установлен и запущен.

Создайте базу данных (например, nasa_db).

Скопируйте файл .env.example в .env и отредактируйте параметры подключения:
```bash
SECRET_KEY=your-secret-key
DEBUG=True
DB_NAME=nasa_db
DB_USER=your_user
DB_PASSWORD=your_password
DB_HOST=localhost
DB_PORT=3306
```
### 5. Применение миграций
```bash
python manage.py migrate
```

### 6. Создание суперпользователя (для доступа в админку) и сбор статических файлов
```bash
python manage.py createsuperuser
python manage.py collectstatic --noinput
```
### 8. Запуск сервера
```bash
python manage.py runserver
```
Сайт будет доступен по адресу http://127.0.0.1:8000.
Админка: http://127.0.0.1:8000/admin

## 🎮 Использование
### Админка
Зайдите в админку по адресу /admin.

В разделе Слайды вы можете добавлять, редактировать и удалять элементы.

При добавлении слайда выберите изображение через виджет django-filer. Если вы впервые загружаете файл, сначала создайте папку (например, «Слайды») внутри интерфейса filer, затем загрузите туда изображения.

Сортировка записей осуществляется перетаскиванием за иконку с тремя полосками слева от строки.

В списке отображается миниатюра изображения и название.

Главная страница
Слайдер автоматически отображает все добавленные в админке слайды.

Используйте миниатюры внизу для навигации или стрелки.

Кликните на большое изображение — откроется полноэкранная галерея, где можно листать все фото.
## 🔧 Особенности реализации
Синхронизация слайдеров — реализована через опцию asNavFor в Slick Slider.

Полноэкранная галерея — модальное окно Bootstrap с классом modal-fullscreen и отдельным слайдером.

jQuery в админке — в admin.py добавлен jquery.init.js, чтобы избежать конфликтов и обеспечить работу виджета filer.

Drag & drop сортировка — модель наследует SortableMixin, в админке используется SortableAdminMixin.

Безопасность — секретные ключи и параметры БД хранятся в .env (пример в .env.example).
